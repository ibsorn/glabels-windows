VERIFICATION
------------

This package embeds no binaries. `chocolateyinstall.ps1` downloads the installer at
install time and Chocolatey verifies it against the SHA-256 checksum in that script
before running it.

Package version 3.99.639 installs:

  https://github.com/ibsorn/glabels-windows/releases/download/win-3.99-master639/gLabels-3.99-master639-win64-setup.exe
  sha256: 670B329289E21A468199262591342EDCAE24E0B8770912CCABA9160E8CC4CA3A

To verify that checksum yourself:

  Invoke-WebRequest -Uri <url above> -OutFile setup.exe
  Get-FileHash setup.exe -Algorithm SHA256

The same sum is published as `SHA256SUMS.txt` on the release itself:

  https://github.com/ibsorn/glabels-windows/releases/tag/win-3.99-master639
